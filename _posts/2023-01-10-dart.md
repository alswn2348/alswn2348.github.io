###  Dart

## 특징

- 다트는 구글에서 제작 했으며 user iterface를 만드는 데에 최적화되어 있다.
- 멀티 플랫폼을 지원한다.
- 생산적인 개발환경을 제공한다.
- 모든 플랫폼에서 빠른속도를 지원한다.

## 컴파일러

Dart 는 dart web 과 dart native두가지 컴파일러를 제공한다.
dart web은 javascript로 변환해주는 컴파일러이다.
dart native 는 거의 모든곳에서 작동하도록 컴파일 할 수 있다.
컴파일 방법은 jit 와 aot 두개를 사용한다.


## 클래스

```dart
class Player{
  String name = 'minju';
  int age = 26;
}

void main() {
  var player = Player();
  print(player.name);
  print(player.age);
}
```
